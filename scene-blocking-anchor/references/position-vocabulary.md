# 相对方位词典

**核心原则**：所有位置描述必须用**相对方位词**，禁用现实方位词，禁用量化坐标。

## 三条铁律

1. **禁用现实方位词**（南门 / 北墙 / 朝东南）—— AI 图像模型看不懂东西南北，但看得懂"画面里那个最显眼物体的反方向"
2. **禁用量化坐标**（横向 50% / 纵向 65% / 身体侧 90°）—— 模型读数字困惑，读"在主锚左侧 2 步"才能拍
3. **必用相对主锚方位词** + 面向关系

## 必用词 vs 禁用词

### 画面位置

| ✅ 必用 | ❌ 禁用 |
| --- | --- |
| 画面左侧 | 中央偏下（模糊） |
| 画面中央 | 附近（无方向） |
| 画面右侧 | 横向 50%（数字） |
| 画面左下 | 主位附近（无参考物） |
| 画面右上 | 靠近画面（无距离） |

### 相对主锚位置（A 模式，场景物体锚）

| ✅ 必用 | ❌ 禁用 |
| --- | --- |
| 在主锚正前方 | 在主锚那边（无方向） |
| 在主锚左侧 | 靠近主锚（无远近） |
| 在主锚右侧 | 主锚周围（模糊） |
| 在主锚身后 | 主锚附近（无距离） |
| 在主锚前方 2 步 |  |
| 在主锚左后方 3 步 |  |

### 相对主角自身位置（B 模式，主角锚）

| ✅ 必用 | ❌ 禁用 |
| --- | --- |
| 在主角左侧 1 步 | 在主角旁边 |
| 在主角右后方 | 跟着主角 |
| 围主角形成半圆 | 主角周围 |
| 在主角前方 2 步 |  |

### 角色之间

| ✅ 必用 | ❌ 禁用 |
| --- | --- |
| 在<对方>正前方 | 在<对方>旁边 |
| 在<对方>左前方 | 跟<对方>站一起 |
| 在<对方>右后方 | 靠着<对方> |
| 在<对方>身后 | 挨着<对方> |

### 面向关系

| ✅ 必用 | ❌ 禁用 |
| --- | --- |
| 面对<对方>（正面） | 跟<对方>站着（无面向） |
| 侧身对<对方>（侧面） | 看着<对方>（只说视线没说身体） |
| 背对<对方> | 朝向<对方>（模糊） |
| 面对主锚 |  |
| 背对主锚 |  |
| 侧对主锚（左肩对镜头） |  |

### 视线

| ✅ 必用 | ❌ 禁用 |
| --- | --- |
| A 平视 B | A 看 B（无角度） |
| A 俯视 B | 对视（没说谁俯谁仰） |
| A 仰视 B | A 望着 B（模糊） |
| A 视线落在 B 左手上 |  |
| A 视线落在<道具>上 |  |

## 8 方位速查表

面对主锚时以主锚为原点的 8 方位：

```
       左后方   身后    右后方
              ↑
   左侧  ←  [主锚]  →  右侧
              ↓
       左前方   正前方  右前方
```

任何角色的位置必须落在这 8 个方位之一（或"正前方 N 步""左侧 1 步"等距离细化）。

## 面向关系速查表

任何两个角色的面向关系必须落在以下 4 类之一：

| 关系 | 描述 |
| --- | --- |
| **正面对视** | A 正面对 B，B 正面对 A |
| **侧面** | A 侧身对 B（左肩 / 右肩对 B） |
| **背对** | A 背对 B（B 在 A 身后） |
| **不共面** | A 面向主锚，B 面向另一方向（不发生视线交流） |

写视线/面向关系表时每对角色必须归入 4 类之一，不允许模糊。

## 绝对方位锁（1 句话兜底）

每个角色除了相对主锚位置，还要有一句"绝对方位锁"，跨 Panel 不漂：

| 字段 | 内容 |
| --- | --- |
| 永远在画面的 | 画面中央偏下 / 画面左侧 / 画面右侧（3 选 1） |
| 永远位于主锚的 | 主锚正前方 / 左侧 / 右侧 / 后方（8 方位选 1） |

这两个字段一旦定了，跨所有 Panel 不能改，除非剧本明写位置变化。
