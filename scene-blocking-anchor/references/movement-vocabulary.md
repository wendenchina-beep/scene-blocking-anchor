# 走位描述词典（三要素齐备）

**核心原则**：每条走位描述必须同时含**朝向 + 距离动词 + 距离刻度**三要素，三要素齐才能锁死画面。

## 为什么三要素齐备

**朝向 ≠ 距离**：
- "背对主锚"只描述**面朝哪边**，不描述**跟主锚的距离**
- "面朝背离主锚方向"仍可能"离主锚越来越近"（如后退走）

**每条走位描述必须同时含**：
1. **朝向**（面朝哪边）
2. **距离动词**（推远 / 拉近 / 收窄 / 拉开距离 / 远离 / 靠近）
3. **距离刻度**（N 步 / 一臂之距 / 半个场景外 / 出画）

三要素齐才能锁死画面。

## 走位类型词典

### 朝主锚走近（距离缩小）

| ✅ 三要素齐 | ❌ 缺要素 |
| --- | --- |
| 朝主锚**走近 N 步**，距离从 M 步 → M-N 步 | 朝主锚走（无距离） |
| 向主锚**靠近**，收窄到一臂之距 | 向前（无参考物） |
| 跟主锚的距离**拉近** | 靠过去（无方向） |

### 离开主锚（距离拉大）

| ✅ 三要素齐 | ❌ 缺要素 |
| --- | --- |
| 朝**背离主锚方向走远 N 步**，距离从 M 步 → M+N 步 | 朝背离主锚走（只有朝向没距离，可能倒退走仍在主锚附近） |
| 跟主锚的**距离拉大** | 向后退（可能仍在主锚附近 or 已出画，歧义） |
| **远离**主锚，直到 N 步外 / 出画 | 走远（无起终点） |
| 向镜头方向走，主锚在画面深处越缩越小 |  |

### 被动移动（拖 / 拽 / 推 / 抬）朝远离主锚方向

| ✅ 三要素齐 | ❌ 缺要素 |
| --- | --- |
| 被**朝背离主锚方向拖走**，距离从 M 步 → 拖出画外 / 拖到 N 步外 | 被拖走（无方向） |
| 被**推远**主锚，距离从一臂之距拉大到画面深处 | 被朝背离主锚方向拖（无距离，可能"背对主锚但没走远"） |
| 被**拽向远处**，主锚在画面里越来越小 |  |

### 被动移动朝靠近主锚方向

| ✅ 三要素齐 | ❌ 缺要素 |
| --- | --- |
| 被**推向主锚**，距离从 N 步 → 主锚正前方 | 被推（无方向） |
| 被**拉近**主锚，直到一臂之距 |  |

### 横向移动（距离不变，角度变）

| ✅ 三要素齐 | ❌ 缺要素 |
| --- | --- |
| 从主锚正前方 → 主锚左侧，**距离保持 N 步** | 侧向走（距离歧义） |
| 绕主锚走半圈，始终 N 步距离 |  |

### 出画 / 出场景

| ✅ 三要素齐 | ❌ 缺要素 |
| --- | --- |
| 走出画面（主锚已看不见 / 主锚在角色身后消失） | 离开（去哪不明） |
| 被拖出场景，从主锚一臂之距 → 完全出画 |  |

## 距离刻度参考

只用"步数 / 臂 / 场景比例"，禁用米/厘米/百分比：

| 距离档 | 描述词 |
| --- | --- |
| 极近 | 一臂之距 / 半步 / 贴身 |
| 近 | 1-2 步 |
| 中 | 3-5 步 / 半个场景内 |
| 远 | 半个场景外 / 6+ 步 / 画面深处 |
| 出画 | 走出画面 / 主锚从视野中消失 / 出场景 |

## 关键写法示例

**场景**：<角色 A> 被拖离主锚

- ❌ **朝向单写**：「A 被**朝背离主锚方向**拖走」→ 模型可能拍出"A 背对主锚但被拖到主锚附近"
- ✅ **朝向 + 距离动词 + 距离刻度**：「A 被**朝背离主锚方向拖走 N 步**，跟主锚的距离**拉大**到画面深处 / 直到出画」→ 模型能锁住"远离主锚"这个语义
- ✅ **末态明确**：「A 被拖到画面前景边缘，主锚已在 A 身后画面深处，距离从一臂之距 → 半个场景外」→ 位置起终点 + 距离变化都清楚

## 常见错误反向示例

**剧本**：<主角>面对灵位跪着 → 被婆子拖走

| ❌ 错误写法（agent 易写错） | ✅ 正确写法（主锚 = 灵位） |
| --- | --- |
| 「主角跪在灵位前，被婆子向南门方向拖走」 | 「主角**面对主锚（灵位）**跪着，被婆子**朝背离主锚的方向**（向镜头方向）拖走 5 步，距离从一臂之距 → 半个场景外」 |
| 「主角被推向灵位」（剧本明明是远离灵位） | 「主角被**朝主锚反方向**（画面前景方向）拖走，远离主锚到画面前景边缘」 |
| 「主角朝北走 / 朝东侧拖」 | 「主角朝**背离主锚的方向**走 / 朝**主锚左侧**移动 3 步」 |

**关键**：「背离主锚 / 朝主锚反方向」比「朝南门 / 朝北墙」AI 准确率高 10 倍，因为 AI 看图能识别"画面里那个最显眼物体的反方向"，但识别不出"南北东西"。
